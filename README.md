Tic Tac Toe Ultimate

A modern Tic Tac Toe game built using HTML, CSS, and JavaScript.

This game includes sound effects, scoreboard, AI opponent, and animated winner screen.

🚀 Features

✅ Play with Friend (Player vs Player)  
✅ Play with Computer (AI Mode)  
✅ Sound Effects (Click, Winner, Background Music)  
✅ Volume Control  
✅ Scoreboard System  
✅ Animated Winner Screen  
✅ Responsive Design  

---

**Technologies Used**

- HTML5
- CSS3
- JavaScript

**How to Play**

1. Choose **Play With Friend** or **Play With Computer**
2. Players take turns placing **X** or **O**
3. First player to align 3 symbols wins
4. Score updates automatically
